## 阅读建议

1. 想了解当前代码已经公开哪些 MCP 工具，先读 [01-当前工具与实现状态](01-当前工具与实现状态.md)。
2. 想看后续进阶能力还剩哪些待做项，读 [02-进阶实现 TODO](02-进阶实现TODO.md)。
3. 想追溯初版设计、状态机、错误模型、测试矩阵和施工记录，再读 [1.0.0文档归档/00-索引.md](1.0.0文档归档/00-索引.md)。
4. 若当前文档与归档文档出现差异，以当前代码和顶层当前文档为准；归档文档保留历史背景和设计证据。

## 维护规则

- 顶层 docs 面向当前状态，只添加主要文档。
- 1.0.0 归档内容不搬回顶层，除非要形成新的当前版本文档。
- 新增工具、改变工具契约或更新验证状态时，优先更新 [01-当前工具与实现状态](01-当前工具与实现状态.md) 与 README。
- 新增或完成进阶能力时，同步更新 [02-进阶实现 TODO](02-进阶实现TODO.md)。
