# 17 初版施工任务拆分清单

本文档承接 [10-开发里程碑](10-开发里程碑.md)、[13-代码施工大纲](13-代码施工大纲.md)、[15-M0-工程骨架施工记录](15-M0-工程骨架施工记录.md) 与 [16-M1-模型与错误基线施工记录](16-M1-模型与错误基线施工记录.md)，把初版 M0 到 M8 拆成可持续施工、可验收、可回滚的 issue 级任务清单。

本文档是任务板，不重新定义 [03-初版工具契约](03-初版工具契约.md)、[05-状态机与资源生命周期](05-状态机与资源生命周期.md)、[06-并发与缓冲区设计](06-并发与缓冲区设计.md)、[07-错误模型与返回格式](07-错误模型与返回格式.md)、[08-测试与验收计划](08-测试与验收计划.md) 或 [09-库选择与依赖决策](09-库选择与依赖决策.md) 已经确定的行为。任务执行中发现设计冲突时，应先回写对应设计文档或施工记录，再继续实现。

## 使用规则

- 任务编号使用 `M阶段-序号`，例如 `M0-01`、`M4-06`。
- 每个任务完成前必须有代码、测试或文档证据，不以“文件已创建”或“函数占位”视为完成。
- 同一阶段内可以并行处理无共享状态的任务，但不得越过前置 gate。
- 阶段完成时必须回填对应施工记录或验收记录；M0 回填 15，M1 回填 16，M8 可在 `docs/acceptance/` 形成最终验收记录。
- 若任务需要改变工具契约、状态机、错误码、并发语义或依赖策略，先更新上游文档，再改代码。

## 状态标记

| 标记 | 含义 |
| --- | --- |
| `Todo` | 尚未开始。 |
| `Doing` | 正在施工，尚未满足完成条件。 |
| `Blocked` | 被前置依赖、设计冲突、依赖问题或环境问题阻塞。 |
| `Done` | 完成条件和验证命令均已满足，并有证据记录。 |

| --- | --- | --- |

## 关键任务卡

### SDK Session Gate

| 字段 | 内容 |
| --- | --- |
| 关联任务 | M0-05、M0-06、M1-05、M7-03 |
| 核心问题 | `rmcp` request context 是否能稳定区分 AI 会话。 |
| 通过条件 | 能稳定获得 session ID 时启用 `session_mode=auto`；不能时实现 `session_mode=unavailable`，并让依赖会话身份的路径返回 `SESSION_ID_UNAVAILABLE`。 |
| 回写位置 | 14、15、16。 |
| 禁止事项 | 不允许在 session 不稳定时偷偷共享全局默认实例或共享订阅队列。 |

### Error Shape Gate

| 字段 | 内容 |
| --- | --- |
| 关联任务 | M1-03、M1-04、M1-05、M2-07、M7-03 |
| 核心问题 | 所有工具成功/失败是否都经统一返回模型输出。 |
| 通过条件 | 失败返回包含 `ok=false`、`tool`、`request_id`、`timestamp`、`error.category`、`error.code`、`recovery_hint`、`retryable` 和脱敏 details。 |
| 回写位置 | 16、07 如有冲突。 |
| 禁止事项 | 不允许 handler 临时拼 JSON 或把外部库原始错误直接返回。 |

### Resource Lifecycle Gate

| 字段 | 内容 |
| --- | --- |
| 关联任务 | M2-05、M2-06、M3-06、M5-02、M5-03、M6-03 |
| 核心问题 | release/force release 后资源锁和底层关闭是否可解释。 |
| 通过条件 | 强制释放后句柄立即失效，资源锁进入 `closing` tombstone，同资源在后台 close 完成或 deadline 前不能复用。 |
| 回写位置 | 05、08、17，必要时补 acceptance 记录。 |
| 禁止事项 | 不允许为了快速重连直接释放仍在关闭的资源锁。 |

### No-Hardware CI Gate

| 字段 | 内容 |
| --- | --- |
| 关联任务 | M3-02、M4-08、M5-01、M5-03、M6-01、M8-01 |
| 核心问题 | 没有串口硬件时 CI 是否仍能稳定验证核心行为。 |
| 通过条件 | MockTransport、TCP/UDP loopback、错误模型、并发、多会话测试不依赖真实串口；串口手工验收单独记录。 |
| 回写位置 | 08、11、12、M8 acceptance。 |
| 禁止事项 | 不允许把真实串口设备作为默认自动化测试前置。 |
